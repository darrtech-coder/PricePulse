Price Tracker Web App

This project is a simple web-based interface for tracking product prices.

Features:
- Add new products (name, supplier, URL)
- Add/edit/delete products
- Manage suppliers with dropdown
- Manual sync button for price updates
- Excel-based history tracking
- Modular for scraper/API logic

Requirements:
- Python 3.x
- pip install -r requirements.txt

How to run:
1. Open a terminal in the project folder.
2. Run: python app.py
3. Open your browser and go to http://localhost:5000

Notes:
- You can integrate scraping or API-based updates later in app.py.
